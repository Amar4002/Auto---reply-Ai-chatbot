# AI WhatsApp Auto-Reply Bot

This project is an automated WhatsApp chatbot built using:
- ✅ Python + PyAutoGUI (Desktop automation)
- ✅ Google Gemini API (AI responses)
- ✅ Clipboard-based chat reading

## Features
- Automatically reads the latest message from WhatsApp Desktop
- Detects sender and avoids replying to self
- Generates human-like Hinglish responses using Gemini AI
- Sends reply back into WhatsApp automatically
